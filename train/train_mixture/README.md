# Train Mixture

This directory contains the training pipeline for a unified mixture dataset.

## Overview
- **Dataset**: `dataset_train_mixture` (S3).
- **Model**: Stable Diffusion v1.5 (Fine-tuning).
- **Objective**: Standard Diffusion.

## Usage
1. Update `config.py` with credentials.
2. Run training:
   ```bash
   ./train.sh
   ```
